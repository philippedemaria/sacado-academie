{% load static %} Aide : Comment enregistrer le fichier CSV en UTF-8 ?
      1. Exporter votre base de données en .csv ou .xlsx ou .ods
      2. Ouvrir Excel ou OpenOffice Calc et copier le fichier dans un nouveau document
      3. Enregister le document avec Enregistrer Sous
      4. Choisir le Type Text CSV (*.csv) et valider. Si une fenetre surgit, conserver le modèle CSV.
      5. La fenetre Export de fichiers texte apparait. Choisir le jeu de caractères Unicode (UTF-8) et le point virgule comme séparateur de champ.
