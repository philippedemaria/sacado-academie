Un flashpack est un ensemble de flashcards (ou cartes mémoires).

Une carte mémoire est une carte sur laquelle vous pouvez écrire une définition, une formule, une méthode à connaitre.

Une fois que votre flashpack est créé, vous pouvez revoir toutes les notions d'un même thème ou chapitre qui vous parait important. Ainsi, vous réactivez votre mémoire pour fixer les notions.

Cette réactivation, ritualisée, est portée par les sciences cognitives et Stanislas Dehaene dont voici son approche par les 4 piliers de l'apprentissage :

  1. L'attention
  2. L'engagement actif
  3. Le retour d’information
  4. La consolidation
