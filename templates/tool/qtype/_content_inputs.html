Utiliser \input[name][3em]{valeur par défaut} pour placer le champ à compléter.
$\frac{\input[num][3em]{}}{\input[deno][3em]{2}}$ affiche : et utilise les 2 variables num et deno.
Pour les champs de réponse dans l'énoncé, écrire dans "réponse attendue" (ci-dessous) les réponses au format name=reponse. num=4 signifie que la réponse du champ num est 4. Ajouter autant de réponses que de variables déclarées.
Si la variable name est le résultat du calcul des variables numériques aléatoires a et b, écrire : num=$?!{a} + {b} ?!$ dans "réponse attendue" (ci-dessous).
Pour inclure une formule, utiliser les délimiteurs simple Dollar $ et pour inclure une figure, utiliser TiKZ
